# WebSocket API Documentation v2

## Connection
Base URL: `/socket.io`

## Authentication
Authentication is handled via JWT token:
- Provide token in the connection query: `?token=your_jwt_token`
- Or in the handshake auth object: `{ auth: { token: 'your_jwt_token' } }`

## Context Events

### Subscribe to Context Events
